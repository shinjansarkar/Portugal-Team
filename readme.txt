Portugal Football Team Website:-

    A simple HTML and CSS project that showcases the entire Portugal football team along with detailed player profiles. This         website allows you to view each player's current club and the national team they represent.

Features:

    Player Profiles: View detailed profiles of each player, including their current club and national team.

    Interactive Layout: The website layout allows easy navigation through the roster, making it user-friendly.

Technologies Used

    HTML5: Structure of the website.
    CSS3: Styling and layout design.

Getting Started
Prerequisites

    A web browser (e.g., Chrome, Firefox, Safari) to view the website.
    Text editor (e.g., VS Code, Sublime Text) if you want to edit the code.
